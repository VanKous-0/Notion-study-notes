# Markdown 学习笔记 (VS Code 版)

## 1. 基础排版 (Basic Formatting)

| **语法** | **代码示例** | **说明** |
| --- | --- | --- |
| **标题** | `# 一级标题`
`## 二级标题` | 井号数量代表层级，最多 6 级。 |
| **粗体** | `**重点内容**` | 强调文本，两边各两个星号。 |
| **斜体** | `*斜体文字*` | 轻微强调，两边各一个星号。 |
| **引用** | `> 引用一段话` | 也就是 Blockquote，用于摘录或强调。 |
| **分割线** | `---` | 也就是 Horizontal Rule，用于分隔章节。 |

## 2. 列表与整理 (Lists)

- **无序列表**：使用减号  或星号  加空格。Markdown
    
    `- 买牛奶`
    
     `- 买面包`
    
- **有序列表**：使用数字加点 `1.` 加空格。Markdown
    
    `1. 第一步`
    
    `2. 第二步`
    
- **任务清单 (Checklist)**：Markdown
    
    `- [ ] 待办事项 (未完成)`
    
     `- [x] 已完成事项`
    

## 3. 链接与多媒体 (Links & Images)

- **超链接**：`[显示的文字](链接地址)`
- **图片**：`![图片描述](图片链接)`
    - *技巧*：VS Code 中可以直接引用本地图片，如 `![截图](images/demo.png)`。

## 4. 代码块 (Code)

- **行内代码**：用反引号 ``` 包裹。
    - 示例：使用 ``git status`` 查看状态。
- **多行代码块**：用三个反引号包裹，并指定语言。
    
    ```python
    
    def hello():
    
    print(”hello world”)
    
    ```
    

## 5. 进阶语法 (Advanced)

### 表格 (Tables)

Markdown

`| 姓名 | 职业 | 技能 |
| :--- | :---: | ---: |
| 蜘蛛侠 | 学生 | 吐丝 |
| 钢铁侠 | 富豪 | 钞能力 |` 

*(注意：冒号 `:` 用于控制对齐方向，左边是左对齐，右边是右对齐)*

### 数学公式 (Math / LaTeX)

- **行内公式**：`$E=mc^2$`
- **块级公式**：Code snippet
    
    `$$
    \sum_{i=1}^{n} i = \frac{n(n+1)}{2}
    $$`
