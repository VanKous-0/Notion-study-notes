## 📝 仓库说明
- 本仓库用于整理我的大学计算机课程笔记  
- 原始内容在 Notion 中编辑，再导出 Markdown 上传到 GitHub  
- 每门课程都放在独立文件夹，结构清晰便于扩展  
- 文件格式主要为 Markdown（.md）与图片资源  

---

## 🚀 使用方式
你可以直接打开对应课程文件夹查看内容。  
Markdown 文件 GitHub 会自动渲染，无需额外工具。

---

## 📌 更新方式（记录给自己）
1. 在 Notion 更新内容  
2. 导出为 Markdown  
3. 覆盖本地对应课程文件夹  
4. `git add .`  
5. `git commit -m "更新某某课程笔记"`  
6. `git push`
